# __ui extension
Define some options for the ui-builder (furoc-gen-u33e).


#### ____ui extension in a field of a type


```yaml
fields:
  id:
    type: string
    description: The identifier.
    __proto:
        number: 1
        oneof: ""
    __ui:
      component: "furo-data-text-input"
      flags: []
      no_init: false
```
